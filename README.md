# Website bán thiết bị điện tử
## I. GIới thiệu chung
Website bán thiết bị điện tử được xây dựng bằng **Microsoft ASP.NET Core MVC**. Đây là website thương mại điện tử mô phỏng việc bán các thiết bị máy tính với đầy đủ chức năng.

## III. Cơ sở dữ liệu
- **Backend**  
  - [ASP.NET Core MVC](https://dotnet.microsoft.com/apps/aspnet/mvc) – Xây dựng ứng dụng web theo mô hình MVC  
  - [Entity Framework Core](https://learn.microsoft.com/ef/core) – ORM để làm việc với database  
  - [SignalR](https://learn.microsoft.com/aspnet/core/signalr) – Giao tiếp thời gian thực (real-time) giữa client và server  

- **Frontend**  
  - [Razor View](https://learn.microsoft.com/aspnet/core/mvc/views/razor) – Engine render giao diện từ server  
  - [HTML5](https://developer.mozilla.org/docs/Web/HTML) – Cấu trúc nội dung website  
  - [CSS3](https://developer.mozilla.org/docs/Web/CSS) – Thiết kế giao diện và responsive layout  
  - [JavaScript](https://developer.mozilla.org/docs/Web/JavaScript) – Xử lý logic phía client  
  - [AJAX](https://developer.mozilla.org/docs/Web/Guide/AJAX) – Gửi/nhận dữ liệu bất đồng bộ mà không reload trang  
  - [Bootstrap](https://getbootstrap.com/) – Framework UI để tạo giao diện responsive, đẹp và nhanh  

- **Khác**  
  - Quản lý session & authentication bằng **ASP.NET Identity** (nếu có)  
  - Tích hợp API / AJAX cho xử lý dữ liệu động
